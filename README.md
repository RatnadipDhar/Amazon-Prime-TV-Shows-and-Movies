# Amazon-Prime-TV-Shows-and-Movies
This project performs an Exploratory Data Analysis (EDA) on a combined dataset of streaming movies, TV shows, and their cast/crew. The goal is to extract meaningful insights about content performance, audience preferences, and industry trends to support data-driven decision-making in content creation.

Key Highlights of this Project:

1. Data Integration & Cleaning: Merged titles and cast/crew datasets, handled missing values, removed duplicates, and standardized data types for analysis.

2. Exploratory Visualizations: Utilized histograms, bar charts, scatter plots, and correlation heatmaps to analyze IMDb score distributions, content production growth, and the relationship between runtime and ratings.

3. Actionable Business Insights: Identified top-performing genres, evaluated the influence of specific actors and directors on content success, and highlighted trends in digital streaming production.
